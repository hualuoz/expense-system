# 费用分类与编码标准

## 分类体系

### 一级分类

| 编码 | 名称 | 说明 |
|------|------|------|
| TRAVEL | 差旅费 | 因公出差产生的各类费用 |
| OFFICE | 办公费 | 日常办公所需的物资和服务 |
| ENTERTAIN | 招待费 | 业务招待相关费用 |
| TRAINING | 培训费 | 员工培训相关费用 |
| COMM | 通讯费 | 通信和网络费用 |
| TRANSPORT | 交通费 | 日常市内通勤交通 |
| OTHER | 其他 | 无法归入以上类别的费用 |

### 二级分类

#### TRAVEL - 差旅费

| 编码 | 名称 | 票据要求 | 备注 |
|------|------|----------|------|
| TRAVEL-TRANSPORT | 交通费 | 机票/车票/发票 | 含飞机、高铁、火车 |
| TRAVEL-LODGING | 住宿费 | 酒店发票 | 受城市等级和职级限制 |
| TRAVEL-MEAL | 餐饮补贴 | 无需票据 | 按城市等级和天数计算 |
| TRAVEL-OTHER | 其他差旅 | 发票 | 如行李托运、签证费等 |

#### OFFICE - 办公费

| 编码 | 名称 | 票据要求 | 备注 |
|------|------|----------|------|
| OFFICE-SUPPLIES | 办公用品 | 发票 | 文具、耗材等 |
| OFFICE-EQUIPMENT | 办公设备 | 增值税专票 | 须经IT部门采购 |
| OFFICE-SOFTWARE | 软件订阅 | 发票 | 须经IT部门审批 |
| OFFICE-PRINT | 印刷费 | 发票 | 名片、宣传品等 |
| OFFICE-POST | 邮寄费 | 发票/快递单 | |

#### ENTERTAIN - 招待费

| 编码 | 名称 | 票据要求 | 备注 |
|------|------|----------|------|
| ENTERTAIN-MEAL | 业务餐费 | 发票 | 须附参与者名单和事由 |
| ENTERTAIN-GIFT | 商务礼品 | 发票 | 须提前审批 |
| ENTERTAIN-OTHER | 其他招待 | 发票 | |

#### TRAINING - 培训费

| 编码 | 名称 | 票据要求 | 备注 |
|------|------|----------|------|
| TRAINING-COURSE | 培训课程 | 发票 | 须提前审批 |
| TRAINING-MATERIAL | 培训教材 | 发票 | |
| TRAINING-CERT | 认证考试 | 发票 | |

#### COMM - 通讯费

| 编码 | 名称 | 票据要求 | 备注 |
|------|------|----------|------|
| COMM-MOBILE | 手机通讯 | 发票/账单 | 月度补贴制 |
| COMM-NETWORK | 网络费用 | 发票 | |

#### TRANSPORT - 交通费（日常）

| 编码 | 名称 | 票据要求 | 备注 |
|------|------|----------|------|
| TRANSPORT-TAXI | 出租车 | 发票 | 须说明事由 |
| TRANSPORT-PARKING | 停车费 | 发票 | |
| TRANSPORT-FUEL | 油费 | 发票 | 须附里程记录 |
| TRANSPORT-PUBLIC | 公共交通 | 卡充值发票 | |

#### OTHER - 其他

| 编码 | 名称 | 票据要求 | 备注 |
|------|------|----------|------|
| OTHER-MISC | 杂项 | 发票 | 须详细说明用途 |
| OTHER-DONATION | 捐赠 | 收据 | 须公司批准 |

## 城市等级划分

### 一线城市
北京、上海、广州、深圳

### 二线城市
成都、杭州、武汉、南京、重庆、天津、苏州、西安、长沙、青岛、郑州、大连、东莞、宁波、厦门、福州、合肥、昆明、哈尔滨、沈阳、济南、无锡、佛山

### 三线及以下
上述以外的城市

## 费用编码使用示例

```
示例1：北京出差高铁票
category: TRAVEL-TRANSPORT
description: 北京-上海高铁二等座
amount: 553.00

示例2：上海出差酒店住宿
category: TRAVEL-LODGING
description: 上海出差住宿（如家酒店，2晚）
amount: 798.00

示例3：客户招待午餐
category: ENTERTAIN-MEAL
description: 与XX公司项目沟通午餐（3人）
amount: 450.00
```
